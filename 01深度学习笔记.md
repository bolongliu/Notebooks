## 01. 对于张量的理解
一般来说看张量是几维，数张量的开头最外层的`[`即可，有几个[就是几维。一般cv用4维张量。[b,c,h,w] b:批量大小，c:通道, h:图像高度，w:图像宽度.例如[10,3,256,256]，表示一个10个样本，通道为3,高宽为256的图像。
下面是一个形状为 (2, 3, 4, 5) 的张量.
```python
import numpy as np

# 创建一个形状为 (2, 3, 4, 5) 的张量，值从 0 到 119
tensor = np.arange(120).reshape(2, 3, 4, 5)
print(2*3*4*5)
# 打印张量（输出格式类似上面的文本表示）
print(tensor)
```
